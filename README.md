# 用途別Djangoのサンプル

- taskmanager_v1
    - celeryを使ったサンプル

## Djangoのコマンド

ユーザーの追加

```bash
python3 manage.py createsuperuser
```

マイグレーションの実行

```bash
python3 manage.py makemigrations
python3 manage.py migrate
```


プロジョクトの作成

```bash
django-admin startproject [project name]
```

アプリの追加

```bash
python3 manage.py startapp [app name]
```

ライブラリのインストール

```bash
pip3 install -r requirements.txt
```
